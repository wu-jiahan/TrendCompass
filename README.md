\f0\fs24 \cf0 # TrendCompass\
\
**Quantifying Cultural Trend Fatigue Through Dual-Signal Intelligence**\
\
Portfolio project by Wu Jiahan \'97 Sungkyunkwan University, B.A. Media and Communication\
\
## Overview\
TrendCompass detects cultural trend fatigue by combining Google Trends search interest \
with NLP-based sentiment analysis of Reddit comments. The system computes a daily \
Fatigue Index (\uc0\u8722 1 to +1) and visualizes it in an interactive Streamlit dashboard.\
\
## Case Studies\
- **Coachella** \'97 event-driven fad lifecycle\
- **Alo Yoga aesthetic** \'97 identity-embedded durable trend\
\
