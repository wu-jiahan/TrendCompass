# TrendCompass

**Quantifying Cultural Trend Fatigue Through Dual-Signal Intelligence**

Portfolio project by Wu Jiahan — Sungkyunkwan University, B.A. Media and Communication

## Overview
TrendCompass detects cultural trend fatigue by combining Google Trends search interest 
with NLP-based sentiment analysis of Reddit comments. The system computes a daily 
Fatigue Index (−1 to +1) and visualizes it in an interactive Streamlit dashboard.

## Case Studies
- **Coachella** — event-driven fad lifecycle
- **Alo Yoga aesthetic** — identity-embedded durable trend

## Tech Stack
Python · pandas · Hugging Face Transformers · Streamlit · Plotly · PullPush.io API

## Theoretical Framework
Rogers (1962) Diffusion of Innovations · Brehm (1966) Psychological Reactance · 
Maslach (1982) Emotional Exhaustion
